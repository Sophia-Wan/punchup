Punch Up Ideation
------------------
Ideation: Where the landing page of my name has small interactions, like when users hover a letter, it shakes, and if they click on some letters too much, the letter may drop, or something hiding behind it will appear like some sort of illustration. The letters would have some butterflies and moon kind of theme following my portfolio, while the clouds follow the colour scheme gradient.

It'll help enhance my work, as the first thing people will see when they click into my portfolio is an interactive title that can be played around with. It will be kind of like a game loading screen where it introduces me instead. I will also create some graphics and illustrations on Adobe Illustrator to add to the things hiding behind letters. Since my portfolio is quite "static" right now, I think having this interactive intro will help draw people in.
